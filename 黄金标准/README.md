# 黄金标准语料库 (Gold Standard Corpus)

## 概述

本目录包含中国博物馆多语种解说词平行语料库的**黄金标准子集**，即中文-英文双语对照语料。

## 黄金标准的定义

本语料库中的中英对照数据具有以下特点：

### 1. 来源权威

- **原文**：来自博物馆展板的官方中文解说词
- **英文**：来自博物馆展板的官方英文翻译（博物馆聘请专业翻译团队完成）
- **非机翻**：英文翻译不是机器翻译，而是人工翻译的专业版本

### 2. 质量保证

- 展板上的中英文对照经过博物馆专业人员审核
- 翻译质量高，术语准确，表达地道
- 符合博物馆解说词的专业标准和语言规范

### 3. 处理流程

```
原始展板照片 → OCR识别 → LLM增强校正 → 人工审核 → 黄金标准
```

- OCR识别提取展板上的中英文文本
- LLM增强校正OCR识别错误
- 人工审核确认最终质量

## 文件说明

### `zh_en_parallel.json`

中英双语平行语料库主文件，包含：

| 字段 | 说明 |
|-----|------|
| `id` | 唯一标识符（博物馆_图片_段落_句子） |
| `source.museum` | 博物馆名称 |
| `source.image` | 来源图片文件名 |
| `source.title_zh` | 展板中文标题 |
| `source.title_en` | 展板英文标题 |
| `translations.zh` | 中文原文 |
| `translations.en` | 英文官方翻译 |

## 与完整语料库的区别

| 特性 | 黄金标准 (本目录) | 完整语料库 (交付/) |
|-----|------------------|-------------------|
| 语言 | 中英双语 | 五语种 (zh/en/ja/es/bg) |
| 翻译来源 | 博物馆官方翻译 | 官方翻译 + 机器翻译 |
| 用途 | 质量评估、翻译基准 | 语料库应用、NLP研究 |

## 黄金标准的价值

1. **翻译质量评估**：可作为机器翻译系统的评估基准
2. **翻译模型训练**：高质量平行语料可用于训练翻译模型
3. **术语库构建**：博物馆专业术语的标准化参考
4. **学术研究**：博物馆解说词语言特征研究
